# Catalog

The catalog contains public, stable descriptions of datasets used by the agents.

For the project overview and replication path, start with the root [README](../README.md). For Agent Builder specs that consume this catalog context, see [specs/README.md](../specs/README.md).

## Contents

- `datasets/` contains reviewed dataset summaries and intended agent usage.
- `field-dictionary/` contains the DMR field dictionary and recommended readable aliases.

Generated cluster snapshots belong in `catalog/raw/`, which is ignored because it may contain private index names or environment details.
